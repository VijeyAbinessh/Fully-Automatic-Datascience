🚀 Auto ML Platform

This platform automates the complete Machine Learning workflow — from data cleaning to model training and resource discovery — using an easy-to-use web interface.

🔧 Available Features

  • 🔍 Resource Finder
    Finds relevant datasets, GitHub repositories, baseline models, and evaluation metrics based on your problem statement.

  • 📊 Auto Data Reporter
    Generates professional analytical reports with statistics, correlations, outlier analysis, and insights — exportable as Word files.

  • 🧹 Auto Data Cleaner
    Automatically cleans uploaded datasets by fixing column names, handling missing values, removing duplicates, and treating outliers.

  • 🤖 Auto Model Trainer
    Train machine learning models by selecting target variables, features, task type, and algorithms directly from the UI. No coding or command-line interaction required.

🧭 How to Use (Quick Guide)

  1. Start with Resource Finder for datasets and baseline research.
  2. Use Auto Data Reporter to understand your data and insights.
  3. Go to Auto Data Cleaner to clean your raw dataset.
  4. Use Auto Model Trainer to train and download ML models.

🎯 Who Is This For?

  • Students & Hackathon Teams
  • Data Science Beginners
  • Researchers & Analysts
  • Rapid ML Prototyping

Goal: Reduce manual effort and make Machine Learning accessible, repeatable, and production-ready.
